This page allows you to provide details of any resources that are related to your collection. This could include publications, physical archives, other digital archives or other online resources. These links will appear alongside your collection once it is published and will allow users to find these resources from our digital repository.

To add a Related Resource, click the “+ Related Resource" button. There are three mandatory fields to complete for each resource:

### Title

This is the title of the associated resource. For a publication or other archive, this could be the overall title of that resource. This is a mandated field.

### Resource Identifier Type

From the drop down list, select one of the resource identifier options, such as DOI (Digital Object Identifier) or URL (uniform resource location, otherwise known as a weblink). For a full list of the Resource Identifier types and what they relate to, please see [Appendix ?](). The actual link to that resource will be included in the Resource Identifier field. This is a mandatory field.

### Resource Identifier

Please enter the identifier for your related resource. This could be a web link (URL) or Digital Object Identifier. Please ensure that the identifier matches with the type listed in the Resource Identifier Type. This is a mandatory field.

Some examples could include:

* __DOI__ - e.g. https://doi.org/10.5284/1234567
* __URL__ - e.g. https://archaeologydataservice.ac.uk/archives/1234567/metadata.cfm
* __ISBN__ -  e.g. 990-0-12-123456-1
