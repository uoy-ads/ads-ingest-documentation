This page allows you to enter details about the People & Organisations that are associated with this collection. This includes those who hold copyright for the collection, those who created it and those who funded it. This page also allows you to enter details about who to contact if there are any questions about your collection. Some of these details will appear on your published collection page.

Some fields in the People & Organisations page can be pre-populated with information provided in the [Settings menu](../gs/gs_settings.md) including Default Copyright Holder and Default Web Contact. This ensures that you do not have to re-enter these details for every collection. The default options can be edited within this page if required.

### Copyright Holders

Please enter details about who is the copyright holder(s) of the collection. This may include the original author(s) of the collection, or the organisation who is responsible for creating the content. If you are working on collaborative projects across different organisations, it is possible to include multiple copyright holders using this menu.

To add details about the copyright holder first select the ‘Type’ \- either a *Person* or *Organisation*. 

If you selected a *Person* you will to complete the following fields:

* __First Name__
* __Last Name__
* __ORCID__ - The sixteen digit reference number associated Open Researcher and Contributor ID ([ORCiD](https://info.orcid.org/what-is-orcid/))

If you selected *Organisation* you will only need to enter the Organisation name. Type in the field to search for available options. If the organisation that you are looking for does not appear, please type in the name manually.

You can select more than one option, either another Person or Organisation, by again clicking the “+ Add Copyright Holder" button. To remove an option, click the delete icon on the right hand side of the entry.

### Data Creators

Please enter the names of individuals or organisations who created the data in the collection. This could represent multiple people or organisations. They could also include fieldworkers, researchers or personnel involved in data entry.

To add details about the data creator first select the ‘Type’ \- either a *Person* or **Organisation*. 

If you selected a *Person* you will to complete the following fields:

* __First Name__
* __Last Name__
* __ORCID__ - The sixteen digit reference number associated Open Researcher and Contributor ID ([ORCiD](https://info.orcid.org/what-is-orcid/))

If you selected ‘Organisation’ you will only need to enter the Organisation name. Type in the field to search for available options. If the organisation that you are looking for does not appear, please type in the name manually.

You can select more than one option by again clicking the “+ Add Data Creator" button. To remove an option, click the delete icon on the right hand side of the entry.

### Project Funders

Please enter details of the organisation(s) who funded the work or research that resulted in this collection. 

To add a Project Funder, click the “+ Add Project Funder" button. Type under ‘Organisation Name’ to search available options. If the organisation that you are looking for does not appear, please type in the name manually.

You can select more than one option by again clicking the “+ Project Funder" button. To remove an option, click the delete icon on the right hand side of the entry.

### Primary Contact

Please enter the name and email address of the person who is the primary contact should an archivist need to contact you about your collection. These contact details will not be made public once the collection is published.

### Web contact

Please enter information about who users should contact if they have a question about your collection. This name and email address will be publicly available alongside your collection when it is published. A Web Contact can be a Person or an Organisation. 

If you are depositing on behalf of an organisation, you may wish to include a general email address rather than the contact details of a specific person.

Please note, if you do enter the details of a specific person under web contact, please ensure you receive their permission to put their contact details on our website prior to submitting your collection.

To add details about the Web Contact first select the ‘Type’ \- either a *Person* or **Organisation*. 

If you selected *Person* you will need to complete the following fields:

* __First Name__ - a mandatory field
* __Last Name__ - a mandatory field
* __ORCID__ - The sixteen digit reference number associated Open Researcher and Contributor ID ([ORCiD](https://info.orcid.org/what-is-orcid/))  
* __Email address__

If you selected *Organisation*, you will need to complete the following fields:

* __Organisation Name__ - Type in the field to search available options. If the organisation that you are looking for does not appear, please type in the name manually.  
* __Email address__
