# Welcome to ADS Ingest

This documentation serves as a comprehensive guide to ADS Ingest, the new digital deposit system for the [Archaeology Data Service](https://archaeologydataservice.ac.uk/). Developed to streamline the process of archiving archaeological digital data, this system represents a significant advancement in our data management capabilities.

ADS Ingest enables researchers, field archaeologists, and heritage professionals to deposit and preserve archaeological datasets through an intuitive interface. The system supports multiple file formats, automated metadata extraction, and seamless integration with existing research workflows.

## This Documentation

This documentation provides detailed instructions for all user types, from first-time depositors to system administrators. Whether you're preparing to archive data from a watching brief or complex excavation dataset these guides will help you navigate the system efficiently.

If you have any recommendations for improvements or additions to this documentation please contact the [ADS Helpdesk](https://archaeologydataservice.ac.uk/contact/) with the subject line 'Ingest Documentation'.
