# Addressing errors

This section covers the errors you may encounter when using the Ingest system, and how to fix them. Whether you're completing the Core Metadata Template or uploading files, most errors have a straightforward cause and a clear resolution — this section is designed to help you find it quickly and continue with your deposit.

---

## How to Use This Section

This part of the guide provides a list of common errors that occur when uploading [Files](../nc/nc_files.md) and [Metadata](../cmt/cmt_index.md) to the Ingest system.

Errors in Ingest fall into two categories. Use the links below to go directly to the relevant table:

| I'm having a problem with... | Go to |
|---|---|
| The Core Metadata Template | [Core Metadata Template Errors](../appendix/core_metadata_errors.md) |
| Uploading files | [File Upload Errors](../appendix/file_upload_errors.md) |

Each table lists the error message you will see in the system, a description of what has caused it, and — where possible — the steps to fix it.

---

## If You Can't Find a Solution

The error tables cover all known errors in the Ingest system. If you encounter an error that isn't listed, or if the suggested fix doesn't resolve your problem:

**Contact the helpdesk**
Email us with a description of the error, the stage of the deposit you were at, and — if possible — a screenshot.
📧 [help@archaeologydataservice.ac.uk](mailto:help@archaeologydataservice.ac.uk)

> When contacting the helpdesk about an error, including the exact error message you see in the system will help us respond more quickly.

**Attend a drop-in session**
If you'd prefer to talk to us in person, please bring your question to one of our Ingest drop-in sessions. Members of the ADS and HSDS team will be on hand to help.
👉 [View drop-in session dates and register](../tr/tr_ingest_qa.md)

---

## Reporting a New Error

If you believe you have encountered a bug or an error that is not covered in this documentation, please contact the Helpdesk and let us know. Your information help us improve the system for all users.

📧 [help@archaeologydataservice.ac.uk](mailto:help@archaeologydataservice.ac.uk)
