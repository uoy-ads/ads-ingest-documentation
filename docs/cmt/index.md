# Introduction

This menu provides instructions for how to complete metadata to go alongside the objects that you would like to deposit in the Ingest system. Accurate and complete metadata ensures that your digital deposit is discoverable, usable, and properly preserved in our archive.

Ingest utilises a single Core Metadata Template for depositing data into Ingest. 

Benefits of this system include:

* A single metadata template for all your files  
* A simpler structure for your metadata records  
* Automatic error checking  
* Easy error rectification

## Instructions

Prior to depositing a New Collection, you should organise the data you would like to deposit and create appropriate metadata. This metadata should be recorded using the [Core Metadata Template](cmt_template_download.md).

Download and complete this single Metadata Template for all of the digital objects that you would like to deposit. Each row of the template will describe a single digital object that forms part of your collection.

Please complete this template prior to starting a New Collection, as this file will be used to validate the files that you are depositing.

!!! note "**Please note**"

    It is important that the details listed in the Core Metadata Template are properly formatted to allow for automatic checking and comparison with the data deposited into the system. Any errors will be highlighted during the deposit process but must be amended before the deposition can proceed.
