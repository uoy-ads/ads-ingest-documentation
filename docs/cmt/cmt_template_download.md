A .xlsx version of the Core Metadata Template can be downloaded from the following link:

<div style="text-align: center;">
  <a href="https://ingest.archaeologydataservice.ac.uk/api/v1/metadata-spreadsheet-template" class="md-button md-button--primary">Download Template</a>
</div>

The template can also be downloaded from the [Files menu](../nc/nc_files.md) when depositing using the Ingest system.

The Core Metadata Template contains example data to assist in completing the form (written in red). Please delete this data before uploading the template to Ingest.
