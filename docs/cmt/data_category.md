The table below lists all of the Data Categories accepted through the Ingest system. This table provides a decription of each data type as well as accepted file extentions and links to Technical Metadata templates, where required.

| Data Category | Accepted File Extensions | Description | Metadata Templates |
| :---- | :---- | :---- | :---- |
| **Audiovisual** | acc, aif, au, avi, bwf, flack, mkv, mp3, mp4, mpg, avg, wav, mpeg | Audio or video files. |  [Audiovisual technical metadata](https://docs.google.com/spreadsheets/d/1FBJuZ_Jo4sLXMB66biyfQvj48xtNm1G4/edit?gid=129668891#gid=129668891) |
| **Computed Tomography (CT) Scanning** | bmp, dcm, gif, jpg, png, tif | Images, models, or data from CT scanning including micro CT. CT video files should be categorised as Audiovisual. |  |
| **Geographic Information System (GIS) Data** | adf, asc, csv, e00, geojson, gml, img, jpg, kml, mid, png, shp, tif | Raster or vector datasets, along with linked tabular data, commonly used within GIS software. | [GIS technical metadata](https://docs.google.com/spreadsheets/d/1ZGL6Uh6-5QEhuUcozOm0GmqVTEdY6sD1/edit?gid=980696367#gid=980696367) |
| **Geophysics** | csv, dat, mgr, rep, sgy, txt, xcp | Instrument data from geophysical survey equipment. Processed raster or vector data (e.g. geotiff or DXF files) should be deposited as GIS Data. | [Geophysical technical metadata](https://docs.google.com/spreadsheets/d/1GPccthZfSrTj2W7aR3QUIhOjgC3BgzbF/edit?gid=2136171927#gid=2136171927) |
| **Geophysics (Proprietary)** | \[any\] | Instrument data from geophysical survey in proprietary (or exclusive) formats. This must be deposited with a version of the data in an accepted (i.e. non-proprietary) deposit format. | [Geophysical technical metadata](https://docs.google.com/spreadsheets/d/1GPccthZfSrTj2W7aR3QUIhOjgC3BgzbF/edit?gid=2136171927#gid=2136171927) |
| **Harris Matrix** | csv, dxf, lst, pdf, txt \[images\] | Harris matrices files that describe the stratigraphic sequence of an archaeological investigation  |  |
| **Laser Scanning** | tif, txt, e57, las, obj, pts | Point data from terrestrial  laser scanning equipment.  Please note that LiDAR derived data should be categorised as LiDAR. |  [Laser scan metadata](https://docs.google.com/spreadsheets/u/0/d/1vT0RK9cm3Xa9pwCgIodt3VRMvStg53akT-cVC1o2Uj4/edit), [3D model metadata](https://docs.google.com/spreadsheets/d/1QXvAkwXjagMCqw04VbUS1rYScp9w1MmVsLsTELH8hF0/edit?gid=0#gid=0), [3D registration/alignment metadata](https://docs.google.com/spreadsheets/d/1KQZ1yXU9Wb3KjIZSbbNYzkEEJwIS3Jwl-hY8y8hlnQ4/edit?gid=0#gid=0), [3D survey control metadata](https://docs.google.com/spreadsheets/d/1luNM082vTtQlSoAXOuq96IpuN_XY-HWs5gO2FY8CpCA/edit?gid=0#gid=0) |
|  **Light Detection and Ranging (LiDAR)** | tif, txt, e57, las, obj, pts | Raw point data captured from aerial or mobile systems. |  LiDAR technical metadata  |
| **Magnetic Resonance Imaging (MRI)** | bmp, dcm, gif, jpg, png, tif \[video\] | Data produced from MRI scanning of artefacts and human remains. |  |
| **Marked Up Text** | html, sgml, xhtml, md | Text and documents formatted using markup or markdown languages. |  |
| **Mass Spectroscopy**  | .jdx, mzdata, mzml, mzxml, txt, xml \[tabular data\] | Data resulting from Mass Spectroscopy techniques, including MS, Raman and XRF. |  Mass spec technical metadata |
| **Other Document**  | doc, docx, odt, pdf, rtf, txt, md | Other Text documents that are not categorised as ‘Publication’, ‘Report’, or ‘Site Records’. |  |
| **Other Raster Image** | bmp, dng, gif, jp2, jpg, pdf, png, tiff  | Other Raster images that are not categorised as ‘Photograph’  |  |
| **Photogrammetric Survey** | dng, jpg, obj, pts, tiff, wrl | Data (including images, outputs and models) from photogrammetric surveys. |  [3D photography camera metadata](https://docs.google.com/spreadsheets/d/1k49t7MtwrVdRRhAjYUBfZWM5hrGmIsOXVhWUbYSp9XM/edit?usp=drive_web&ouid=101016921283803964772), [3D photography calibration metadata](https://docs.google.com/spreadsheets/d/10i_o3k5okJF7mQNFCIxVa8Lg9sTElsRjA0hlCaIdWR0/edit?gid=0#gid=0), [3D model metadata](https://docs.google.com/spreadsheets/d/1QXvAkwXjagMCqw04VbUS1rYScp9w1MmVsLsTELH8hF0/edit?gid=0#gid=0), [3D survey control metadata](https://docs.google.com/spreadsheets/d/1luNM082vTtQlSoAXOuq96IpuN_XY-HWs5gO2FY8CpCA/edit?gid=0#gid=0), [3D registration/alignment metadata](https://docs.google.com/spreadsheets/d/1KQZ1yXU9Wb3KjIZSbbNYzkEEJwIS3Jwl-hY8y8hlnQ4/edit?gid=0#gid=0) |
| **Photograph** | bmp, dng, gif, jp2, jpg, pdf, png, tiff | Digitally captured or scanned photographs. |   |
| **Publication** | doc, docx, odt, pdf, rtf, txt, md | Text files that are formal publications (not including Reports). |  |
| **Radiocarbon** | xml, svg oxcal, 14c,prior, js, log, txt (oxcal input and output) pdf \[tabular/data bases\] | Data and dates from radiocarbon dating. |  Radiocarbon technical metadata |
| **Reflectance Transformation Imaging (RTI)** | dng, jpg, obj, ptm, rti, wrl, tif,   | Data (images and outputs) derived from the photographic method of RTI. |  |
| **Report** | doc, docx, odt, pdf, rtf, txt, md | Text files that are formal reports Text files that represent a digital version of a formal publication, whether online or in print. |  |
| **Site Record** | doc, docx, odt, pdf, rtf, txt, md, plus all the images  | Scanned physical site records such as context and other recording sheets, site registers, site plans and section drawings. |  |
| **Tabular Data** | accdb, csv, dbf, json, mdb, odb, ods, rdf, sql, tsv, txt, xls, xlsx, xml | Tabular datasets such as databases and spreadsheets. |  [Tabular technical metadata](https://docs.google.com/spreadsheets/d/1p1HGIr3uD0xsAHCeztaBGrl8AAgzCkmK/edit?gid=1325703402#gid=1325703402) |
| **Vector Graphic** | dwg, dxf, svg | Graphical vector datasets such as CAD (computer assisted design) models. |  [Vector technical metadata](https://docs.google.com/spreadsheets/d/1BX2YB74S7FLCAtHuR6nlla9zyJfSzyEo/edit?gid=1479513331#gid=1479513331) |
| **Virtual Reality and Visualisation** | obj, stl, wrl  | Other 3D models and visualisation including virtual reality. This data should not include datasets derived from ‘Photogrammetry’. ‘Laser scanning’, ‘LIDAR’ or derived from specific medical imaging techniques such as CT Scanning. |  [3D model metadata](https://docs.google.com/spreadsheets/d/1QXvAkwXjagMCqw04VbUS1rYScp9w1MmVsLsTELH8hF0/edit?gid=0#gid=0) |
| **X-radiography (X-ray) Scanning** | bmp, dcm, gif, jpg, png, tif  | Images and data produced by X-ray scanning. |  |
